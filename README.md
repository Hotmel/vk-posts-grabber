# VKPostsGrabber

Ретрансляция постов из ВКонтакте в Telegram. Теперь заходить в ВК вовсе не обязательно, все в любимом телеграме!

## Установка

```
$ git clone https://github.com/bifot/VKPostsGrabber
$ cd VKPostsGrabber
$ npm install
$ node app.js
```
